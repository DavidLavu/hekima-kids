# Butterfly Maths 🦋

An offline maths game for young children (UK Years 2–3): answer questions, catch British butterflies, fill your album.

Plain HTML/CSS/JS — `index.html` plus `css/` and `js/` — no build step, no dependencies, no accounts, no data collection. Open `index.html` in any browser (works from a `file://` URL). Progress lives in the browser via localStorage.
